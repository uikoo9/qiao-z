# qiao-z

nodejs 下极简的 web 框架

## 官网

[https://qiao-z.insistime.com/#/](https://qiao-z.insistime.com/#/)

## 安装

```shell
npm i qiao-z
```

## 使用

```javascript
// qz
const qz = require('qiao-z');

// app
const app = qz();

// listen
app.listen();
```

## 示例

```shell
git clone ...
cd packages/qiao-z/__tests__
node app.js

# open http://localhost:5277
```
