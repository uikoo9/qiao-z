# qiao-z
nodejs下极简的web框架

## 官网 
[https://code.insistime.com/qiao-z](https://code.insistime.com/qiao-z)

## 安装
```bash
npm i qiao-z
```

## 使用
```javascript
// qz
const qz = require('qiao-z');

// app
const app = qz();

// listen
app.listen();
```

## 示例
```bash
git clone ...
cd packages/qiao-z/__tests__
node app.js

# open http://localhost:5277
```
